Decentralized Messaging

A fully decentralized messaging platform built on Ethereum. Send and receive messages from Ethereum addresses and ENS domains.

Start free 14 days trial

Open Source

Clone the repo and use docker to start everything you need to parse your own messages and view them with no middlemen involved, or to implement custom actions when you receive a message.

Secure

All messages sent using the platform are encrypted with the receiver's key, so only the intended recipients read your messages.

Multipurpose

dApp Owners can communicate with their users to let them know of big announcements, or send them reminders. Users can communicate P2P to discuss a trade or share ideas.
